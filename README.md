# Vue 3 + TypeScript + Vite

使用 vite+element-plus 搭建项目模板
node 版本 > 16+

## 运行步骤

### 安装依赖

npm install

### 运行

npm run dev

## 使用的插件

element—plus

### 使用的插件 bin-dataV

bin-dataV 大屏插件
https://wangbin3162.gitee.io/bin-datav/#/guide

### 加载进度条 nprogress

npm install --save nprogress
